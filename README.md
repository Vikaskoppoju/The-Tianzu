Explore our indian culture with us
INDIAN CULTURE
'Indian Culture' is an web application, which describes the culture and tradition of Indian States individually in an easy way.
This web application gives an user-friendly environment with a map and individual buttons to get the details of particular state. Homepage of this web application contains the detailed information about Indian culture and the improvement of culture from ancient times to modern along with a map, which enables user to select the respective state to get detailed information about it. And list of state buttons to redirect users to a webpage which contains detailed information about the state.
The technologies used for this web application are HTML, CSS ,JS for frontend and Python for backend logic and SQLite3 for database. All these are connected through famous web framework called Django.
Once, terminal/cmd is opened from the location of manage.py run the following command 'python manage.py runserver'. Then open browser and enter the following URL 'http://127.0.0.1:8000/home/'.
The list of modules required to run this web application are
- Django
